# このサイトについて

エンジニアとして仕事を進めていくにあたってなにか困った時に、ブログ記事など事象単位での解決法はありがたいことにたくさんあります。が、エンジニアの仕事というフローとしての知見はあまりないのかな？と思ったので、自分なりにまとめてみることにしました。基本的には自分用の備忘録ではありますが、プロジェクトメンバなどと意志の共有をするための助けになるとよいなと思っています。

## 筆者について

都内でフロントエンドエンジニアとして勤務しています。詳細は<a href={SITE.portfolio}>ポートフォリオサイト</a>を作っていますので、そちらを参照いただけると嬉しいです。

## 全体/一般セクションについて

体調管理の仕方、仕事一般で気をつけていること、採用についてなど、技術的な部分以外についてつらつら書いていこうかと思います。

## このサイトの使用技術など

- [Astro](https://astro.build/)、TypeScript、MDX、Tailwind CSS
- ホスティングはNetlify
- リポジトリは[こちら](https://github.com/kzk4043/knowledge-doc)

## 注意事項

- このサイトは筆者の個人的な見解を自由気ままに書いています。会社の見解ではありませんので、ご注意ください。
- 自分向けの備忘録という意味合いが強いため、わかりにくい部分があるかもしれませんが、ご容赦ください。
